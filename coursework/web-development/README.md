# Web Development

This directory contains coursework related to **front-end web development**,
focusing on building structured, responsive, and maintainable web pages using
modern HTML and CSS practices.

The coursework demonstrates fundamental web design principles including layout
structuring, styling consistency, and responsiveness across different screen sizes.

---

## 📁 Directory Structure

```text
web-development/
├── assignments/
├── labs/
└── README.md
```

---

## 🧪 Labs

The **labs** focus on practicing core web development concepts through smaller,
targeted exercises.

Topics covered include:

* HTML document structure
* Semantic elements
* CSS selectors and properties
* Layout fundamentals
* Styling techniques
* Page organization best practices

Each lab is implemented with emphasis on:

* Readable and organized code
* Proper indentation and formatting
* Correct use of HTML elements
* Maintainable CSS structure

---

## 📝 Assignments

The **assignments** apply multiple concepts together to build more complete
web pages.

Key focus areas include:

* Responsive layout design
* CSS Grid and Flexbox usage
* Visual consistency and alignment
* Cross-browser compatibility
* Clean and structured project organization

Assignments include screenshots demonstrating layout behavior across
multiple viewport sizes:

* 375px (mobile)
* 768px (tablet)
* 1200px (desktop)

---

## 🛠 Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Responsive Design Principles

## 🎯 Learning Objectives

Through this coursework, the following skills are demonstrated:

* Structuring web pages using semantic HTML
* Styling pages using modular and readable CSS
* Creating responsive layouts for different screen sizes
* Organizing front-end projects clearly and logically
* Applying fundamental UI layout principles

# Web Development Assignments

This directory contains coursework assignments focused on applying
front-end web development concepts to build structured and responsive web pages.

Each assignment demonstrates the ability to combine HTML and CSS concepts
to produce clean, organized, and visually consistent layouts.

---

## 📚 Topics Covered

Assignments may include:

* Semantic HTML page structure
* Responsive layout design
* CSS Grid and Flexbox
* Consistent spacing and alignment
* Typography and visual hierarchy
* Maintainable and readable CSS
* Cross-browser compatibility

---

## 📱 Responsive Design

Assignments are tested across multiple viewport sizes to ensure
proper layout adaptation and usability on different devices:

* 375px (mobile)
* 768px (tablet)
* 1200px (desktop)

Screenshots are included where required to demonstrate layout behavior
at different screen widths.

---

## 🛠 Technologies Used

* HTML5
* CSS3
* CSS Grid
* Flexbox
* Responsive design techniques

---

## 🎯 Objective

The purpose of these assignments is to demonstrate:

* Understanding of modern layout techniques
* Ability to structure web pages clearly
* Writing clean and maintainable front-end code
* Applying responsive design principles
* Organizing project files logically

# Coursework

This directory contains academic coursework completed as part of the
**Software Engineering** program.

The coursework is organized by subject, with each subject focusing on
specific skills, tools, and problem-solving techniques relevant to
software development and computer science.

---

## 📚 Subjects Included

### Mobile Programming

Focuses on developing Android applications using Java and XML, covering:

* User interface design
* Activities, Fragments, and navigation
* Intents and data passing
* Integration with device features and services

Each lab and assignment is documented with a dedicated README describing
the objective, implementation, and results.

---

### Algorithms Analysis

Focuses on understanding, implementing, and analyzing classic algorithms,
with emphasis on correctness and efficiency.

Topics include:

* Brute Force
* Divide and Conquer
* Transform and Conquer
* Exhaustive Search
* Hashing
* Dynamic Programming

All labs are implemented in Java and documented with detailed explanations
and verified test results.

---

### Web Development

Focuses on building responsive and well-structured web pages using HTML and CSS,
with emphasis on clean layout design and cross-device compatibility.

Topics include:

* Semantic HTML structure
* Responsive layout techniques
* CSS Grid and Flexbox
* Styling consistency and readability
* Cross-browser compatibility

Assignments demonstrate:

* Proper page structure and organization
* Responsive design for multiple screen sizes
* Maintainable and clearly structured styling
* Screenshots showing layouts at 375px, 768px, and 1200px widths
